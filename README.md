# JinyUI Web Component for Laravel
지니UI는 HTML, UI 컴포넌트를 자동으로 생성할 수 있는 객체지향 서버코드를 제공합니다. 제공되는 서버 코드는 HTML 시만틱 테크로 Alias되어 일반적인 HTML UI 코딩이 가능합니다.

## 사용법
지니 UI에 대한 사용법은 /resource/doc 폴더를 참고해 주세요.
